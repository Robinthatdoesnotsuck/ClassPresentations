Collapsible Sidebar Demo

This is a sidebar similar to Claude.ai's interface. Click the toggle button to collapse and expand the sidebar.

The sidebar smoothly animates in and out, with the toggle button moving along with it. The hamburger icon transforms into an X when the sidebar is collapsed.

Key features:

  • Smooth CSS transitions
  • Animated hamburger/X icon
  • Responsive design (fixed sidebar on mobile)
  • Keyboard accessible
  • Clean, modern styling

The sidebar state is saved to localStorage, so it will remember your preference when you reload the page.
